# GoofyGram(Instagram Clone)

This repository contains the code for an Instagram clone built using JavaScript. The frontend is developed with Chakra UI and React, while Firebase is used for the backend services, including authentication, database, and storage.

## Table of Contents

- [Introduction](#introduction)
- [Features](#features)
- [Technologies Used](#technologies-used)
- [Usage](#usage)

## Introduction

This project is an Instagram clone that mimics basic functionalities of Instagram such as user authentication, posting images, liking posts, and following other users. The goal is to provide a comprehensive example of a full-stack web application using modern web development technologies.

## Features

- User authentication (sign up, log in, log out)
- Create, view, and delete posts
- Like and unlike posts
- Follow and unfollow users
- Real-time updates using Firebase

## Technologies Used

- **Frontend:**
  - [React](https://reactjs.org/)
  - [Chakra UI](https://chakra-ui.com/)

- **Backend:**
  - [Firebase Authentication](https://firebase.google.com/products/auth)
  - [Firebase Firestore](https://firebase.google.com/products/firestore)
  - [Firebase Storage](https://firebase.google.com/products/storage)


## Usage:
  -**Visit the site:**
    - visit : (https://goofygram.vercel.app/)

  -**Create an Account or Log In:**
    Use the sign-up form to create a new account or log in with existing credentials.
  
  
  -**Explore the Features:**

    -Create Posts: Click on the "New Post" button to upload images and create new posts.
    -View Posts: Browse through the feed to see posts from all users.
    -Like Posts: Click the heart icon to like a post.
    -Follow Users: Visit a user's profile and click "Follow" to see their posts in your feed.
    -Real-time Updates: Enjoy real-time updates for likes, follows, and new posts thanks to Firebase Firestore.
